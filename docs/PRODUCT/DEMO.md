# Demo assets

Use this checklist when recording a portfolio demo.

- Live app: [https://pulse-notes.vercel.app/](https://pulse-notes.vercel.app/)

## Suggested flow (3–5 minutes)

1. Landing page and sign-up.
2. Create a workspace and a note from a template.
3. Open the same note in a second browser (or incognito) as another user: show presence, typing, and a comment with `@mention` plus the notifications bell.
4. Show workspace **Members** (invite link copy, optional second account accepting `/join/...`), **Templates** tab, and **Invites** tab.
5. Briefly open the workspace **Whiteboard** and note **Preview** (Markdown) mode.

## Artifact placeholders

- **Screen recording:** add your exported MP4 or hosted Loom URL here after you record.
- **GIF (optional):** keep under a few MB for README embedding; link out for HD video.

Example line after recording:

- Demo video: `https://example.com/your-demo.mp4`
